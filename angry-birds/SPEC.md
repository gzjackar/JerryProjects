# 愤怒的小鸟 - 游戏规格说明

## 核心玩法
- 玩家用弹弓发射小鸟，击毁猪猪和防御工事
- 每关3只小鸟，发完则过关或失败
- 击毁所有猪猪即可过关

## 技术实现
- HTML5 Canvas + JavaScript
- 物理引擎：Matter.js (CDN)
- 触控/鼠标支持
- 响应式设计（适配手机屏幕）

## 游戏元素

### 小鸟 (Bird)
- 红色小鸟：标准球形，重量适中
- 黄色小鸟：加速技能（点击加速）
- 蓝色小鸟：分裂成3个小鸟

### 猪猪 (Pig)
- 绿色猪猪：标准目标
- 不同大小，不同耐摔程度

### 材料
- 木头：易碎
- 石头：较坚固
- 玻璃：中度坚固

## 关卡设计
- Level 1：教学关，2只猪
- Level 2：简单破坏
- Level 3：多层结构

## 操作方式
1. 拖拽弹弓橡皮筋
2. 松开发射
3. 飞行中点击触发技能

## 验收标准
- 手机上可触控发射
- 小鸟物理飞行
- 猪猪被击中会消失
- 3个关卡可玩
- 得分系统
